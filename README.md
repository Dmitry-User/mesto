# Проект: Место

Интерактивная страница с возможностью:
 * редактировать профиль
 * добовлять, удалять фотографии
 * ставить лайки

![Mesto](./../images/preview.jpg)

### Технологии
* HTML5
* CSS3
* Nested BEM
* Flexbox
* Grid Layout
* Javascript
